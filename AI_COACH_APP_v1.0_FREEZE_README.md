AI코치 앱 v1.0 봉인 선언문
1. 프로젝트 개요

AI코치 앱은
문제 인식 → 질문 기반 사고 → 요약 → 실행 계획 → 회고 → 기록 저장
의 흐름을 통해 사용자의 자기 성찰과 행동 변화를 돕는 코칭형 학습 애플리케이션이다.

본 버전(v1.0)은
기본 코칭 흐름이 안정적으로 작동하는 최소 기능 제품(MVP)으로 정의한다.

3. 사용자 여정 (User Flow)
홈 → 강의 선택 → 문제 입력 → 질문 답변 → 요약 → 실행 계획 → 회고 → 저장 → 보관함

이 흐름이 끊김 없이 작동하면 v1.0은 성공으로 판단한다.

4. 기술 구조 (Architecture)
src/
  pages/
  routes/
  services/
  utils/
  data/

pages: 화면 단위 구성

routes: 화면 연결

services: 질문/요약 로직

utils: 저장/검색/세션 처리

data: 코칭 커리큘럼 정의

이 구조는 향후 기능 확장을 위한 기반으로 봉인한다.

5. 데이터 구조 원칙

기록은 아래 요소를 포함한다.

lessonId

lessonTitle

problem

answers[]

summary

actionPlan

reflection

createdAt

이 구조는 v1.x 동안 유지한다.

6. UX 원칙

입력이 없는 상태에서는 진행하지 않는다.

질문 흐름은 단순하고 직관적이어야 한다.

저장 후 즉시 결과를 확인할 수 있어야 한다.

검색은 실제 사용 가능한 수준이어야 한다.

7. v1.0 성공 기준

다음이 가능하면 v1.0은 완료로 본다.

전체 코칭 흐름 1회 완주 가능

기록 저장 및 재접속 후 유지

검색 및 삭제 정상 작동

콘솔 에러 없음

화면 깨짐 없음

8. v1.0 봉인 원칙

기능 추가보다 안정성을 우선한다.

구조 변경은 v1.1 이후로 미룬다.

질문 커리큘럼 확장은 별도 버전으로 관리한다.
